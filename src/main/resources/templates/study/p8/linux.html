文件操作

    文件上传: scp 本地文件路径 linux登录名@ip:linux文件夹路径

    tar解压: tar xf ***.tar.gz

    
网络操作

    重新获得ip:
        dhclient
        ifconfig ens33 down
        ifconfig ens33 up
    重启网卡
        1: nmcli c reload
        2: nmcli c up ens33 / nmcli d reapply ens33 / nmcli d connect ens33 三选一

    
yum

    yum install 报错 failed to download metadata for repo 'AppStream'
    	vi /etc/yum.repos.d/CentOS-Base.repo
        修改baseurl=https://mirrors.aliyun.com/centos/$releasever/BaseOS/$basearch/os/

    安装make
        yum -y install gcc automake autoconf libtool make
    安装gcc
        yun install gcc
    安装wget
        yum install wget 可下载网络资源

    
配置环境变量

    vi /etc/profile
        末尾追加
            export REDIS_HOME=/opt/soft/redis6
            export PATH=$PATH:$REDIS_HOME/bin
        source /etc/profile --使配置生效
        echo $PATH 查看path
